# Bank-Management-System
A menu-driven Bank Management System built using Python functions, loops, lists, and conditional statements.
Bank Management System

A beginner Python project that simulates basic banking operations through a menu-driven interface.

Features

- Secure PIN authentication (3 attempts)
- Deposit money
- Withdraw money
- Check account balance
- View transaction history
- Input validation for deposits and withdrawals
- Exit option

Concepts Used

- Variables
- Functions
- Conditional Statements
- Loops (for, while)
- Lists
- User Input Handling
- Basic Program Structure

How It Works

1. User enters a 4-digit PIN.
2. User gets 3 attempts to enter the correct PIN.
3. After successful login, a banking menu is displayed.
4. User can:
   - Deposit money
   - Withdraw money
   - Check balance
   - View transaction history
   - Exit the system

Sample Menu

===== BANK SYSTEM =====

1. Deposit

3. Check Balance
4. History
5. Exit
